# <img src="./images/logo.png" width="30"> Thème DSFR pour Apache Superset

Superset DSFR est une extension de l'outil de datavisualisation Apache Superset. Elle contient tous les éléments de permettant d'appliquer la charte graphique de l'Etat sur ses tableaux de bord !

## Structure du projet
> [!IMPORTANT]  
> Ce projet n'est pas un fork du repo officiel de Superset. Il contient tous les éléments de configuration et de personnalisation de Superset.

``` bash
.
├── .github/ # CI github
├── guides  # Guides de déploiement et cas d'usages
├── images # images de démonstration du repo
├── superset-dsfr # Contient tous les éléments de personnalisation de Superset
├── CHANGELOG.md
├── Dockerfile
├── LICENCE
├── package.json
├── package-lock.json
└── README.md
```
